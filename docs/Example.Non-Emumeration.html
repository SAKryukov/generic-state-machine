Non-Enumeration Example

Source code

The example demonstrate the case of using a non-enumeration type for the generic STATE type of a state machine.

StateMachine<double> stateMachine = new(double.NaN);

The type double has 6 public static fields: NegativeInfinity, MinValue, Epsilon, MaxValue, PositiveInfinity, and NaN. They are used as the states.

Note that the default double value is not usable, because this is 0, not corresponding the any of the double public static fields. Therefore, the call to the constructor specifies NaN as an initial state.

State transition graph:

−∞ ⇔ MinValue ⇔ Epsilon ⇔ MaxValue ⇔ +∞;
MinValue ⇔ MaxValue
NaN ⇒ to any other state, but all the transition to NaN are invalid
